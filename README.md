# langflow-service

langflow-service installer for Linux

Please visit the [Langflow repository](https://github.com/langflow-ai/langflow) for more information.

</br>

### List of tested distribution:
- Debian 13

<br/>

## How to Install

Copy & paste the below command in the terminal:
```
bash -c "$(wget -qLO - https://raw.githubusercontent.com/KimYoungNo/langflow-service/refs/heads/main/langflow-service-install.sh)"
```

## Procedure

After executing the command above, the next procedures automatically begin:

1. System package updates & upgrade
1. Dependency installation
1. Langflow installation
1. langflow-service configuation
1. Service unit creation & enabling
